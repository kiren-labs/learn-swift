# FinChronicle

An iOS application built with SwiftUI.

## Features

- CodeBreaker game functionality
- Match Markers
- Financial chronicle tracking

## Requirements

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+

## Installation

1. Clone the repository
```bash
git clone <repository-url>
cd FinChronicle
```

2. Open the project in Xcode
```bash
open FinChronicle.xcodeproj
```

3. Build and run the project (⌘ + R)

## Project Structure

```
FinChronicle/
├── FinChronicleApp.swift          # App entry point
├── CodeBreaker.swift              # CodeBreaker logic
├── CodeBreakerView.swift          # CodeBreaker UI
├── MatchMarkers.swift             # Match markers functionality
└── FinChronicleViews/
    └── FinChronicleMainView.swift # Main view
```
